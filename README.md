# Blackjack

Blackjack is the famous game of 21! 

## Requirements
- Maven 3.x
- Java 8

## Usage

### Package
`` make package `` or `` mvn clean package`` 

### Install
`` make install `` or `` mvn clean install``

### Run
`` make run `` or `` mvn exec:java -Dexec.mainClass="org.example.blackjack.web.MainWeb"``

[Play!](http://localhost:8080)
